Basic Network and Database Security

2022-01-27

Completed IBM’s course Basic Network and Database Security on EdX.

This was really exciting for me, since it is the first cybersecurity related online course I have completed independently. It was tricky to make time for it everyday- juggling college classes, clubs, and other responsibilities- but overall incredibly rewarding. I’d like to continue to fit independent study within my schedule.

Here are some of the main concepts covered:

  • Stateless inspection and stateful inspection
  • OSI model layers
  • Address Resolution Protocol (ARP)
  • Intrusion detection system (IDS) and intrusion prevention system (IPS)
  • Network address translation
  • Ethernet and LAN-ethernet operations
  • Transport control protocol (TCP) and user datagram protocol (UDP)
  • Domain name system (DNS) and dynamic host configuration protocol (DHCP)
  • Clustering and high availability systems
  • Types of data and how to secure databases
  • OS command injection and prevention tactics
  • SQL injection and prevention tactics
