CySA+ Certification

2023-09-10

Passed CySA+! For two months, I biked to the office an hour early so I could fit in a solid study session before the start of the workday. Since I’m currently on co-op as an Information System Security Analyst, I felt that obtaining this certification would be appropriate. Though there was a lot of content overlap with Security+, it delved deeper into threat detection and analysis, vulnerability management, and incident response. Some topics covered proved to be directly useful in my job functions.

Now onto the next!

Here are some of the main concepts covered:

  • Reconnaissance and intelligence gathering
  • Designing a vulnerability management program
  • Analyzing vulnerability scans
  • Analyzing indicators of compromise
  • Infrastructure and security controls
  • Cloud security
  • Identity and access management security
  • Software and hardware development security
  • Security operations and monitoring
  • Building an incident response program
  • Performing forensic analysis and techniques
  • Containment, eradication, and recovery
  • Risk management
  • Policy and compliance
