CompTIA CySA+ Certification

12-17-2023

Passed CySA+! For a few months, I biked to the office an hour early so I could fit in a study session before the start of the workday. Since I’ve been on co-op as a Cybersecurity Analyst, I felt that obtaining this certification would be an appropriate reinforcement of the skills and knowledge I’m learning. Though there was a lot of content overlap with Security+, it delved deeper into threat detection and analysis, vulnerability management, and incident response. Some topics covered proved to be directly useful in my job functions.

I’ve enjoyed learning about the defensive side of cybersecurity, but going forward I hope to delve deeper into offensive security.

Here are some of the main concepts covered:

  • Reconnaissance and intelligence gathering
  • Designing a vulnerability management program
  • Analyzing vulnerability scans
  • Analyzing indicators of compromise
  • Infrastructure and security controls
  • Cloud security
  • Identity and access management security
  • Software and hardware development security
  • Security operations and monitoring
  • Building an incident response program
  • Performing forensic analysis and techniques
  • Containment, eradication, and recovery
  • Risk management
  • Policy and compliance
