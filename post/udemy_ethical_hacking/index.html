Udemy: Complete Ethical Hacking Course

2022-12-01

Completed The Complete Ethical Hacking Course on Udemy.

This was a solid review of introductory cybersecurity topics and technical skills. While I didn’t learn many new concepts, I enjoyed coding the smaller projects alongside the course, like programming a MAC changer, network scanner, packet listener, and keylogger.

Here are some of the main topics covered:

  • Network/system/website pentesting
  • Wireless attacks
  • Cross site scripting
  • SQL injection
  • Social engineering
