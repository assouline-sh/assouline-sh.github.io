Ongoing    CTF Club Challenges
      As Co-Founder and VP of CTF club, I make challenges for meetings and competitions,       ranging in difficulty and topic to introduce all skill levels to various cybersecurity areas.

01-2024   Sniffer | Python
      Sniff unencrypted credentials sent through less secure protocols by poisoning a target’s       ARP cache and redirect all traffic through the attacking system.

12-2023   Shadow Deletion | Bash
      Replace ‘rm’ to log and keep a hidden copy of a file or directory being deleted. Intended       to retain evidence when an attacker tries to clean up their presence on a system.

11-2023   Collegiate Penetration Testing Competition Report | English
      Executive report written for the 2023 Collegiate Penetration Testing Competition.       Conducted penetration testing on an enterprise network for 8 hours.

08-2023   Linux Security Dashboard | Python
      CLI dashboard of real-time insights on system performance and network metrics. Assess       and monitor system and network health of a Linux system at a glance.

05-2023   Netcat Tool | Python
      Replicated basic Netcat functionality with a tool to execute file uploads, commands, or       an interactive shell on a remote target system.

01-2023   Pwned Password | Bash
      Check if a password has been compromised by submiting part of a given password’s       hash to Have I Been Pwnd? This avoids sending the site a plaintext password.

04-2022   XKCD Password Generator | Bash
      Generate a secure, memorable password using the method described in an XKCD comic.
