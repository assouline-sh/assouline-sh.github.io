Personal Projects

Ongoing

CTF Club Challenges

As Co-Founder and Vice President of the CTF club on campus, I make CTF challenges for our general meetings and competitions. The challenges range in topic and difficulty, and are designed to introduce members of all skill levels to various cybersecurity areas.

01-2024

Sniffer

Python. Sniff a victim’s network traffic by poisoning their ARP cache to redirect all traffic through the attacking system. Looks specifically for unencrypted credentials sent on ports that often run less secure protocols. This project allowed me to explore the powerful ‘Scapy’ library, and the technical implementation of an ARP poisoning attack.

12-2023

Shadow Deletion

Bash. Replace ‘rm’ binary with this script to log a file or directory before it is deleted, and keep a copy of its contents in a hidden directory. Intended to retain evidence when an attacker tries to clean up their presence on a system.

11-2023

Collegiate Penetration Testing Competition Report

English. Report written for the 2023 Collegiate Penetration Testing Competition. I extensively studied offensive security tools and techniques, and gained valuable experience in how to conduct a pentest of a simulated corporate network and deliver an executive report.

08-2023

Linux Security Dashboard

Python. Designed a user-friendly CLI dashboard to display real-time insights on system performance, network metrics, and overall security posture of a Linux system. I wanted to simplify the assessment and monitoring of system and network health, and explore features of the ‘rich’ library to enhance console output.

05-2023

Netcat Tool

Python. Replicated basic Netcat functionality with a tool to execute file uploads, commands, or an interactive shell on a remote target system. Especially useful when Netcat is not on a system, but Python is. I was able to hone my understanding of network protocols and socket programming.

03-2023

Automated Buffer Overflow

Python. Scripted a buffer overflow by creating and delivering precise payloads to blindly exploit a vulnerable TCP service on a remote system. When successful, the script establishes a reverse shell connection and allows for arbitrary code execution. This project deepened my knowledge of the stack frame layout, memory allocation, and low-level system vulnerabilities. It was key to explore debugging techniques and examine assembly language, registers, and memory dumps.

01-2023

Pwned Password

Bash. Use the website Have I Been Pwnd? to check if a password has been compromised by submitting part of a given password’s hash. This avoids sending the site a plaintext password.

04-2022

XKCD Password Generator

Bash. Generate a secure, memorable password using the method described in an XKCD comic. User can specify the number of words to be concatenated, and how many capital letters, numbers, and symbols are in the generated password. First time using ‘argparse’ library.
