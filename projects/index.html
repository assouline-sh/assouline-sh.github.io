Linux Security Dashboard

Python. Designed a user-friendly CLI dashboard to display an overview of a Linux system’s current status and network activity at a glance. I wanted to simplify the monitoring and management of system and network health to facilitate determining one’s security posture and ease system administration.

2023-05

Netcat Tool

Python. Replicated basic Netcat functionality with a tool to execute file uploads, commands, or an interactive shell on a remote target system, especially when Netcat is absent but Python is available. I was able to hone my understanding of network protocols and socket programming.

2023-03

Automated Buffer Overflow

Python. Scripted a buffer overflow by creating and delivering precise payloads to blindly exploit a vulnerable TCP service on a remote system. When successful, the script establishes a reverse shell connection and allows for arbitrary code execution. This project deepened my knowledge of the stack frame layout, memory allocation, and low-level system vulnerabilities. It was key to explore debugging techniques and examine assembly language, registers, and memory dumps.

2023-01

Pwned Password

Bash. Using the website Have I Been Pwnd?, check if a password has been compromised by submitting part of a given password’s hash. This avoids sending the site one’s plaintext password.
